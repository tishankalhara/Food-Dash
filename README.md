# Food Dash 

Food Dash is a full-stack e-commerce web application designed for seamless food ordering and delivery management. It features a modern user interface, dynamic cart logic, and a secure backend management system.

## Features

* **User Authentication & Profiles:** Secure login, registration, and role-based access control.
* **Interactive Shopping Cart:** Real-time adding, updating, and removing items with seamless checkout flows.
* **Admin Dashboard:** Manage food items, track student activities/proctors, and view system statistics.
* **Modern & Responsive UI:** Built with sleek components and intuitive layouts.

---

##  Tech Stack

### Frontend (Client)
* **Framework:** React.js / Vite
* **Styling:** Tailwind CSS

### Backend
* **Runtime Environment:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB
